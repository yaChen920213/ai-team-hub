# AI Team Hub

AI团队创建和成员管理系统

## 项目简介

本项目用于管理 AI 团队的创建、成员管理和协作流程。

## 目录结构

```
ai-team-hub/
├── members/          # 成员信息
├── roles/            # 角色定义
├── projects/         # 项目管理
├── docs/             # 团队文档
└── .github/          # GitHub 配置
```

## 快速开始

1. 添加团队成员信息到 `members/` 目录
2. 定义团队角色到 `roles/` 目录
3. 创建新项目到 `projects/` 目录

## 团队规范

- 所有成员提交代码时请遵循 Commit Message 规范
- 定期更新个人状态和项目进度
- 重要决策请在 Issues 中讨论
