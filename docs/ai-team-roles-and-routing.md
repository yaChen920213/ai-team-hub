# AI Team Roles & Routing

更新时间：2026-03-09

这份文档用来明确当前多实例团队的角色分工、任务边界和路由规则。

---

## 当前团队成员

### 1. Form
**定位：** 品牌 / 包装 / 电商设计

**负责内容：**
- 品牌视觉系统
- 包装设计方向
- 电商主图 / 详情页 / 活动视觉
- 版式、配色、层级、风格统一
- 设计方案判断与优化

**适合丢给 Form 的任务：**
- 这个包装方向成立吗
- 帮我看这套品牌视觉哪里不统一
- 这张电商主图为什么不吸引人
- 给我一个更高级的设计修改方案
- 这组页面视觉层级怎么重做

**不优先给 Form 的任务：**
- 长篇文案撰写
- 社媒口播内容
- 视频镜头语言
- 音频氛围设计

---

### 2. Wit
**定位：** 品牌 / 包装 / 电商文案 / 新媒体文案

**负责内容：**
- 品牌语言系统
- 包装文案
- 电商卖点文案
- 详情页文案结构
- 新媒体内容表达
- 标题、slogan、短文案、转化型表达

**适合丢给 Wit 的任务：**
- 这个品牌怎么说人话
- 帮我写一版包装文案
- 电商详情页卖点怎么排
- 小红书 / 公众号 / 短视频口播文案怎么写
- 这段文案为什么不打动人

**不优先给 Wit 的任务：**
- 视觉版式调整
- 摄影构图判断
- 视频剪辑节奏
- 声音设计

---

### 3. Lens
**定位：** 平面 / 摄影 / 视频 / 音频

**负责内容：**
- 平面视觉判断
- 摄影构图、光线、氛围
- 视频镜头语言、节奏、转场
- 音频质感、空间感、氛围设计
- 把抽象感觉翻译成视听执行建议

**适合丢给 Lens 的任务：**
- 这张图为什么没有高级感
- 这条视频为什么节奏拖
- 帮我拆这个片子的镜头语言
- 这个内容该怎么拍
- 这个视频应该配什么样的声音质感

**不优先给 Lens 的任务：**
- 品牌战略推演
- 电商转化文案主体撰写
- 包装文案主输出
- 通用行政任务

## 三者的关系

### Form / Wit / Lens 的区别

**Form** 关心：
- 视觉系统是否成立
- 设计表达是否统一
- 画面与版式是否专业

**Wit** 关心：
- 话怎么说
- 信息怎么组织
- 表达是否清楚、有力、可传播、可转化

**Lens** 关心：
- 画面怎么被看到
- 镜头和声音如何推动情绪
- 氛围和节奏是否成立

一句话概括：

- **Form**：负责“怎么设计”
- **Wit**：负责“怎么表达”
- **Lens**：负责“怎么感受”

当前状态：Lens 已正式上线，Discord 已连通，可参与正式任务分工。

- 电商视觉
- 视觉系统搭建

- 卖点梳理

- 视频

## 混合任务怎么分

### 情况 1：包装项目
- 视觉结构 / 形式 → **Form**
- 包装正背文案 / 卖点 / slogan → **Wit**
- 产品质感图 / 场景氛围图 / 拍摄建议 → **Lens**

### 情况 2：电商详情页
- 页面风格 / 层级 / 视觉模块 → **Form**
- 卖点文案 / 标题 / 购买理由 → **Wit**
- 视频素材节奏 / 口播氛围 / 音乐建议 → **Lens**

### 情况 3：品牌传播内容
- 品牌主视觉 / KV / 传播画面 → **Form**
- 品牌口号 / 传播文案 / 社媒内容 → **Wit**
- 广告片 / 拍摄 / 短视频 / 声音设计 → **Lens**

## 任务分配优先级

遇到一个任务时，先判断它的主问题属于哪一层：

1. **视觉形式问题** → Form
2. **语言表达问题** → Wit
3. **视听氛围问题** → Lens

如果一个任务跨多层，就按：
- 先主问题
- 再拆给其他实例补位

不要一上来三个人一起上，不然容易重复劳动。

## 推荐使用方式

### 单实例模式
问题边界明确时，直接找对应实例。

### 串联模式
复杂任务可按顺序流转：

- **Form** 定设计方向
- **Wit** 补文案表达
- **Lens** 补拍摄 / 视频 / 音频氛围

### 主控模式
如果你自己还没想清楚该找谁，先把任务丢给主实例，由主实例分流。
